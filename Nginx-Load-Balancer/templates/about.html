About This Project

This web application is built using several technologies to provide a seamless and efficient experience.

Technologies Used

  • Gin Framework: A fast web framework for Go (Golang), used for building the backend API.
  • HTML: The standard markup language used to structure the content on the web page.
  • CSS: Used for styling the web page to make it visually appealing.
  • Nginx: A high-performance web server used to serve the application and manage requests.
  • Docker: A platform used to create, deploy, and run applications inside containers, ensuring consistent environments across different systems.
  • Docker Compose: A tool used to define and run multi-container Docker applications, allowing for easy setup and management of the application's services.
  • Go (Golang): A statically typed, compiled programming language designed for scalability and efficiency, used for backend development in this project.
  • OpenSSL: A toolkit for Secure Sockets Layer (SSL) and Transport Layer Security (TLS) protocols, used here to generate the self-signed SSL certificates for secure HTTPS communication.
